# AI OCR Text Recognition Project

This project is a simple Artificial Intelligence OCR application built using Python.

## Features

- Image upload
- OCR text extraction
- GUI interface
- Text recognition using Tesseract OCR

## Technologies Used

- Python
- OpenCV
- PyTesseract
- Tkinter

## How to Run

Install dependencies:

pip install -r requirements.txt

Run project:

python app.py

## Project Output

The application extracts text from uploaded images and displays it in the GUI.
