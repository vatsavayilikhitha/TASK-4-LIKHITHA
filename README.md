# AI OCR Text Recognition Project

## Overview

This project is a simple Artificial Intelligence OCR (Optical Character Recognition) application developed using Python.

The application allows users to upload an image containing text and automatically extracts the text using OCR technology.

A GUI (Graphical User Interface) is also included using Tkinter.

---

## Features

- Upload image files
- Extract text from images
- GUI interface
- OCR text recognition
- Simple and beginner-friendly AI project

---

## Technologies Used

- Python
- OpenCV
- PyTesseract
- Tkinter
- NumPy

## Installation

Install required libraries:

pip install -r requirements.txt

---

## Run the Project

Run the application using:

python app.py

---

## How It Works

1. User uploads an image
2. OpenCV processes the image
3. PyTesseract extracts text
4. Extracted text is displayed in GUI

---

## Output

The application successfully detects and displays text from uploaded images.
